# GameCoreEvent.CityProjectCompleted
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!example] Example
>
> **Events.CityProjectCompleted**( *playerID*, *cityID*, *projectID*, *buildingIndex*, *iX*, *iY*, *bCancelled* )
* **playerID**    number
* **cityID**    number
* **projectID**    number
* **buildingIndex**    number
* **iX**    number
* **iY**    number
* **bCancelled**

CityProjectCompleted exists in GameEvents. It also exists in Events.
**Firaxis has used CityProjectCompleted with only Events.**
