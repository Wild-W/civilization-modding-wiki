# GameCoreEvent.ResourceVisibilityChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.ResourceVisibilityChanged**( *iX*, *iY*, *resourceType*, *visibilityType* )
* **iX**    number
* **iY**    number
* **resourceType**
* **visibilityType**

ResourceVisibilityChanged exists in GameEvents. It also exists in Events.
**Firaxis has used ResourceVisibilityChanged with only Events.**
