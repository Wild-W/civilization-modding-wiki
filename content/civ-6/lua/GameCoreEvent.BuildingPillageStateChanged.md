# GameCoreEvent.BuildingPillageStateChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **GameEvents.BuildingPillageStateChanged**( *playerID*, *cityID*, *buildingID*, *bPillageState* )
* **playerID**    number
* **cityID**    number
* **buildingID**    number
* **bPillageState**    boolean

BuildingPillageStateChanged exists in GameEvents. It doesn't exist in Events.
**Firaxis has used BuildingPillageStateChanged with only GameEvents.**
