# GameCoreEvent.UnitMovementPointsChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!example] Example
>
> **Events.UnitMovementPointsChanged**( *playerID*, *unitID*, *MovementPoints* )
* **playerID**    number
* **unitID**    number
* **MovementPoints**    number

UnitMovementPointsChanged exists in GameEvents. It also exists in Events.
**Firaxis has used UnitMovementPointsChanged with only Events.**
