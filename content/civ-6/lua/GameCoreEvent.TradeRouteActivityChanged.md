# GameCoreEvent.TradeRouteActivityChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.TradeRouteActivityChanged**( *playerID*, *OriginPlayerID*, *OriginCityID*, *TargetPlayerID*, *TargetCityID* )
* **playerID**    number
* **OriginPlayerID**    number
* **OriginCityID**    number
* **TargetPlayerID**    number
* **TargetCityID**    number

TradeRouteActivityChanged exists in GameEvents. It also exists in Events.
**Firaxis has used TradeRouteActivityChanged with only Events.**
