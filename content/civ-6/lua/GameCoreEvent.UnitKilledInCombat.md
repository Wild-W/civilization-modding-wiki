This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

> **Events.UnitKilledInCombat**( *killedPlayerID*, *killedUnitID*, *playerID*, *unitID* )
* **killedPlayerID**    number
* **killedUnitID**    number
* **playerID**    number
* **unitID**    number

UnitKilledInCombat exists in GameEvents. It also exists in Events.
