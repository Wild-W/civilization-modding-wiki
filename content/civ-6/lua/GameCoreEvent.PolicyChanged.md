# GameCoreEvent.PolicyChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **GameEvents.PolicyChanged**( *playerID*, *policyID*, *bEnacted* )
* **playerID**    number
* **policyID**    number
* **bEnacted**

PolicyChanged exists in GameEvents. It doesn't exist in Events.
**Firaxis has used PolicyChanged with only GameEvents.**
