This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

>
> **GameEvents.CityConquered**( *newPlayerID*, *oldPlayerID*, *newCityID*, *iCityX*, *iCityY* )

* **oldPlayerID**    number
* **newCityID**    number
