# GameCoreEvent.BuildingChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.BuildingChanged**( *iX*, *iY*, *buildingID*, *playerID*, *iPercentComplete*, *iUnknown* )
* **iX**    number
* **iY**    number
* **buildingID**    number
* **playerID**    number
* **iPercentComplete**    number
* **iUnknown**    number

BuildingChanged exists in GameEvents. It also exists in Events.
**Firaxis has used BuildingChanged with only Events.**
