# GameCoreEvent.CityWorkerChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **Events.CityWorkerChanged**( *ownerPlayerID*, *cityID*, *iX*, *iY* )
* **ownerPlayerID**    number
* **cityID**    number
* **iX**    number
* **iY**    number

CityWorkerChanged exists in GameEvents. It also exists in Events.
**Firaxis has used CityWorkerChanged with only Events.**
