# GameCoreEvent.OnPillage
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **GameEvents.OnPillage**( *playerID*, *unitID*, *bImprovement*, *buildingType* )
* **playerID**    number
* **unitID**    number
* **bImprovement**
* **buildingType**

OnPillage exists in GameEvents. It doesn't exist in Events.
**Firaxis has used OnPillage with only GameEvents.**
