# GameCoreEvent.UnitChargesChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.UnitChargesChanged**( *playerID*, *unitID*, *newCharges*, *oldCharges* )
* **playerID**    number
* **unitID**    number
* **newCharges**    number
* **oldCharges**    number

UnitChargesChanged exists in GameEvents. It also exists in Events.
**Firaxis has used UnitChargesChanged with only Events.**
