# GameCoreEvent.OnCityPopulationChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **GameEvents.OnCityPopulationChanged**( *cityOwner*, *cityID*, *ChangeAmount* )
* **cityOwner**
* **cityID**    number
* **ChangeAmount**

OnCityPopulationChanged exists in GameEvents. It doesn't exist in Events.
**Firaxis has used OnCityPopulationChanged with only GameEvents.**
