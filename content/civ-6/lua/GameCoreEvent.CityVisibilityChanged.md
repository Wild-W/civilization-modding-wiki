# GameCoreEvent.CityVisibilityChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **Events.CityVisibilityChanged**( *playerID*, *cityID*, *eVisibility* )
* **playerID**    number
* **cityID**    number
* **eVisibility**    number

CityVisibilityChanged exists in GameEvents. It also exists in Events.
**Firaxis has used CityVisibilityChanged with only Events.**
