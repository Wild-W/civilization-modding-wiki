# GameCoreEvent.EndTurnBlockingChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!example] Example
>
> **Events.EndTurnBlockingChanged**( *ePrevEndTurnBlockingType*, *eNewEndTurnBlockingType* )
* **ePrevEndTurnBlockingType**    number
* **eNewEndTurnBlockingType**    number

EndTurnBlockingChanged exists in GameEvents. It also exists in Events.
**Firaxis has used EndTurnBlockingChanged with only Events.**
