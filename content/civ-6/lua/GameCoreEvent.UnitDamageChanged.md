# GameCoreEvent.UnitDamageChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **Events.UnitDamageChanged**( *playerID*, *unitID*, *newDamage*, *prevDamage* )
* **playerID**    number
* **unitID**    number
* **newDamage**    number
* **prevDamage**    number

UnitDamageChanged exists in GameEvents. It also exists in Events.
**Firaxis has used UnitDamageChanged with only Events.**
