# GameCoreEvent.CitySiegeStatusChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.CitySiegeStatusChanged**( *playerID*, *cityID*, *bIsBesieged* )
* **playerID**    number
* **cityID**    number
* **bIsBesieged**    boolean

CitySiegeStatusChanged exists in GameEvents. It also exists in Events.
**Firaxis has used CitySiegeStatusChanged with only Events.**
