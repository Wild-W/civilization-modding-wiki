# GameCoreEvent.CityProductionChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!example] Example
>
> **Events.CityProductionChanged**( *playerID*, *cityID*, *productionID*, *objectID*, *bCancelled* )
* **playerID**    number
* **cityID**    number
* **productionID**    number
* **objectID**    number
* **bCancelled**

CityProductionChanged exists in GameEvents. It also exists in Events.
**Firaxis has used CityProductionChanged with only Events.**
