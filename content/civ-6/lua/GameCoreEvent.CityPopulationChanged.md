# GameCoreEvent.CityPopulationChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **Events.CityPopulationChanged**( *playerID*, *cityID*, *cityPopulation* )
* **playerID**    number
* **cityID**    number
* **cityPopulation**    number

CityPopulationChanged exists in GameEvents. It also exists in Events.
**Firaxis has used CityPopulationChanged with only Events.**
