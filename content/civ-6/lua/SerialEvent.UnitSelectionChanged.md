# SerialEvent.UnitSelectionChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE]
> **Events.UnitSelectionChanged**( *hexI*, *playerID*, *hexJ*, *isSelected*, *unitID*, *hexK*, *IsEditable* )
* **hexI**    number
* **playerID**    number
* **hexJ**    number
* **isSelected**    boolean
* **unitID**    number
* **hexK**    number
* **IsEditable**    boolean

UnitSelectionChanged exists in GameEvents. It also exists in Events.
**Firaxis has used UnitSelectionChanged with only Events.**
