# GameCoreEvent.DistrictDamageChanged
This is an [Event](civ-6/lua/Events.md). Please refer to that page for more information on Events

## Usage
> [!EXAMPLE] Example
> **Events.DistrictDamageChanged**( *playerID*, *districtID*, *damageType*, *newDamage*, *oldDamage* )
* **playerID**    number
* **districtID**    number
* **damageType**
* **newDamage**    number
* **oldDamage**    number

DistrictDamageChanged exists in GameEvents. It also exists in Events.
**Firaxis has used DistrictDamageChanged with only Events.**
